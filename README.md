# Book-e

O **Book-e** é um WebApp voltado para estudantes universitários, criado para otimizar a organização acadêmica, planejamento de tarefas, acompanhamento de prazos e definição de metas personalizadas. A ferramenta busca aumentar a autonomia e o controle sobre o percurso acadêmico, oferecendo uma solução prática e acessível que integra funcionalidades específicas do ambiente universitário.

---

## Funcionalidades

* **Organização de estudos:** Planejamento de tarefas, criação de metas e acompanhamento de prazos acadêmicos.
* **Acompanhamento de desempenho:** Visualização de desempenho em disciplinas, cálculo de médias e progresso acadêmico.
* **Integração de rotinas de estudo:** Permite centralizar tarefas e estudos, oferecendo uma visão completa do percurso acadêmico.
* **Interface intuitiva:** Foco na usabilidade, permitindo que o aluno visualize seu desempenho e organize seus estudos de forma prática.
* **Personalização:** Definição de prioridades e metas adaptadas ao perfil e necessidades de cada estudante.

---

## Tecnologias Utilizadas

* **Frontend:** HTML, CSS e JavaScript
* **Backend:** Java
* **Banco de Dados:** H2 (versão beta)

---

## Objetivo

O objetivo do Book-e é fornecer uma ferramenta prática, de fácil utilização e especificamente adaptada às necessidades de estudantes universitários. O sistema permite que o aluno:

* Visualize seu desempenho acadêmico de forma clara.
* Organize seus estudos e tarefas de maneira eficiente.
* Tome decisões conscientes sobre suas prioridades e metas acadêmicas.

---

## Como Rodar o Projeto

1. **Clonar o repositório:**

   ```bash
   git clone https://github.com/seu-usuario/book-e.git
   ```
2. **Configurar o backend Java**
   Certifique-se de ter o Java instalado e configure o servidor local conforme as instruções do projeto.
3. **Rodar o banco de dados H2:**
   O banco de dados H2 está configurado em modo beta para testes locais.
4. **Abrir o frontend:**
   Abra o arquivo `login.html` no navegador para acessar a interface do Book-e.
